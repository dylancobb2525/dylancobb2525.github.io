CSCE242: Web Design

Dylan Cobb

This is a 200 level web design class where we design websites using various coding languages. We learn the fundamentals of HTML, CSS, and other web technologies to create functional and visually appealing websites.

GitHub Repository: https://github.com/dylancobb2525/dylancobb2525.github.io/tree/master/csce242-fall2025

Assignments

Assignment 1 - Basic HTML

This assignment focuses on creating a basic HTML webpage about planets. We learn fundamental HTML structure and elements, while building an informative page about planets in our solar system.

Assignment 2 - Basic CSS

This assignment introduces CSS styling techniques. We enhance the HTML page with visual styling, colors, typography, and responsive design principles. This page is specific to wolves.

Assignment 3 - Page Layout

This assignment focuses on advanced page layout techniques using Flexbox. We create complex, responsive layouts with proper grid systems that adapt to different screen sizes and device orientations.

Assignment 4 - Recreate CSS page

Recreate the University of South Carolina McCausland College of Arts and Sciences website using HTML and CSS as well as flexbox.

Assignment 7 - Intro to Java

Introduction to JavaScript with interactive elements including color picker, dynamic text display, and image changes using arrow functions and constants.

Assignment 8 - JS Conditionals

JavaScript conditional statements with plant watering tracker using range slider and digital clock with real-time updates and responsive navigation.

Assignment 9 - Loops

Dynamic scene creation using JavaScript loops, Date object for time detection, and CSS art for clouds, trees, sun, and moon with responsive design.

Assignment 10 - Arrays

Interactive dog adoption gallery using JavaScript arrays, associative arrays, loops, hover effects, and popup functionality with responsive design.

Assignment 11 - Classes

Art gallery with JavaScript classes, modal dialogs, and responsive design showcasing famous paintings with artist information.

Setup Assignment 2: Server Side HelloWorld

Server-side development setup with Node.js and Express. This assignment covers basic server configuration and deployment to Render.
